Sample Code and Libraries

Try out the following sample code and libraries for development using the Europeana API. Share your sample code, libraries, and wrappers by emailing us at api@europeana.eu

europeana4j
This Java library was initially developed by Digibis and subsequently integrated into Europeana's own source code. It provides a clean Java interface to the Europeana API, with JavaBeans encapsulation, transparent pagination of results and utilities to copy and verify thumbnails.
https://github.com/europeana/europeana-client
Europeana Search
Basic Python wrapper for the Europeana API.
https://pypi.python.org/pypi/europeana-search
Django Europeana
This Django library is developed by Justinas Jaronis, Inga Pliavgo, Ignas Bacius, Mykolas Baranauskas from the Lithuanian IT company AKS and provides template tags for easy Europeana content integration.
https://pypi.python.org/pypi/django-europeana
Rest Easy
This Python module dynamically creates wrappers for RESTful APIs from YAML markup. It has a ready-made wrapper for the Europeana API as well as for a number of other GLAM APIs.
https://github.com/reklaklislaw/rest_easy
Europeana PHP-library
This is a PHP-library for the Europeana API. The library is used to power an interactive documentation site for the Europeana API.
https://github.com/dan-nl/europeana-api
Wandora's Europeana Extractor
Wandora is a general purpose information extraction, management and publishing application based on Topic Maps and Java.
http://www.wandora.org/wiki/Download
