Accessibilità

Le funzioni e i contenuti di Europeana.eu sono stati progettati per essere universalmente accessibili agli utenti persone, agli utenti software e ai dispositivi hardware. E’ nostra intenzione soddisfare le Linee guida per l’accessibilità ai contenuti del Web (WC3), Versione 1.0.

Our User Centred Design team frequently conducts usability testing of Europeana.eu navigation, flow, labels, and consistency. Furthermore, we will produce a version – post this prototype phase – for blind individuals.

Alcune delle funzioni di accessibilità al nostro sito sono:

  • titoli di pagina significativi per facilitare l’orientamento
  • Alt text (testo equivalente) significativo per quasi tutte le immagini
  • uso minimo di frame
  • uso di caratteri ridimensionabili
  • uso dei fogli di stile CSS (Cascading Style Sheets) per separare i contenuti dalla presentazione
  • notifica dell’utente quando un link apre una nuova finestra
  • funzionalità (essenziali) utilizzabili senza JavaScript.

Per inviare commenti o domande sull’accessibilità di Europeana.eu, utilizzare il modulo di feedback.
