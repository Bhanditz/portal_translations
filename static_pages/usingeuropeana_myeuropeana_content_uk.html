Використовуючи Мою Європіана

Моя Europeana це ваш персональний простір на Європіана.

Зареєструватись у Моя Europeana and you can:

  • Save this search: Будьте в курсі вибраної теми в Європіана зберігши пошук, так що ви можете легко виконати його знову в майбутньому.
  • Add a tag: Label your favourite items in Europeana with tags that mean something to you. You can then see all objects you have tagged, sorted by the keywords you have chosen, in the My Europeana interface.
  • Save to My Europeana: Збережіть ваші улюблені об'єкти для подальшого просмотру.
  • API ключі: If you are a developer who has registered for one or more API keys, these will be shown on the API Keys tab.

Щоб зберегти пошук, додати таг або зберігти об'єкт, лише увійдіть у систему Моя Europeana and click on the relevant action from the menu.

To view your saved items, searches and tags, use the Моя Europeana shortcuts in the header.
